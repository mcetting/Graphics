Program Description

This project is about making a GC(generalized Cylinder) using vertexes and indicies then drawing them to the screen. Its possible to save your GCs and load them using the html buttons. Just click around to spawn cylinders. The lines that are used as the spine are not displayed to the screen but are stored in the code to build the GC from. Clicking SaveSOR will save the GC as an OBJ file which can be loaded onto the canvas by using the extractSOR button the the right of the saveSOR button.
