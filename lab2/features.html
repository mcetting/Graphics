Program Description

The program prints cylinders that are connected to the screen and does flat shading on them.

The rubberbanding has been reimplemented. You can load the colored cylinder from the files like in the previous assignment.

When you change the cylinders color with my button it makes it unable to display normals so if you want to see the normals just run the default.

Extra Stuff: button that changes between perspective and orthographic, button that changes the clear color, button that changes the color of the cylinders, and finally better html organizing.
